# CDCS Squadron 客户端

## 概述

CDCS Squadron 客户端是一个命令行工具，用于与 CDCS Squadron 服务器进行交互，执行文件筛选、归档、重传和修改等操作。

## 功能

- **健康检查**：检查服务器运行状态
- **设置配置**：设置目标路径和备份路径
- **归档文件**：将指定时间之前的文件归档到备份路径
- **筛选文件**：根据患者ID和时间条件筛选文件
- **重传文件**：将指定文件重新发送到目标路径
- **修改文件**：修改文件中的患者ID并重新发送
- **回滚操作**：回滚之前的重传或修改操作

## 安装

1. 确保已安装 Go 1.24.2 或更高版本
2. 进入客户端目录：
   ```bash
   cd apps/cdcs-squadron/cdcs-co
   ```
3. 安装依赖：
   ```bash
   go get .
   ```
4. 编译客户端：
   ```bash
   go build -o cdcs-co.exe
   ```

## 使用

1. 确保 CDCS Squadron 服务器正在运行（默认端口 8080）
2. 运行客户端：
   ```bash
   cdcs-co.exe
   ```
3. 按照菜单提示选择操作

## 操作说明

### 1. 健康检查

检查服务器的运行状态，显示服务器的状态、运行时间、Go版本和主机名。

### 2. 设置配置

设置目标路径和备份路径：
- 目标路径：扫描XML文件的目录
- 备份路径：归档文件的存储目录

默认值：
- 目标路径：`d:\de`
- 备份路径：`d:\deBackup`

### 3. 归档文件

将指定时间之前的所有文件归档到备份路径，并按照年份和月份进行存储。

需要输入归档时间，格式为 `2006-01-02 15:04:05`。

### 4. 筛选文件

根据条件筛选文件，支持以下条件：
- 患者ID：精确匹配
- 时间筛选：
  - `equal`：等于指定时间
  - `before`：早于指定时间
  - `after`：晚于指定时间
  - `range`：在时间范围内

### 5. 重传文件

将指定文件重新发送到目标路径，新文件命名为 `R当前时间_测试时间.xml`。

### 6. 修改文件

修改文件中的患者ID并重新发送到目标路径，新文件命名为 `M当前时间_测试时间.xml`。

### 7. 回滚操作

回滚之前的重传或修改操作，需要指定操作类型和目录编号。

## 注意事项

- 客户端默认连接到 `http://localhost:8080` 的服务器
- 确保服务器正在运行，否则客户端将无法连接
- 操作过程中请确保网络连接正常
